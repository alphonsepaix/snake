# Snake

![License](https://img.shields.io/badge/license-MIT-blue)

The snake game written with the Bevy engine.

![Gameplay](gameplay.gif)

## Controls

Use the arrow keys (`↑`, `↓`, `←`, `→`) to move around. Press `␣` to pause/unpause the game and `Esc` to exit.
